# To Remember
- 동적계획법의 조건
  - Overlapping Subproblem : 겹치는 부분(작은) 문제
  - Optimal Substructure : 최적 부분구조
    - 문제의 정답을 작은 문제의 정답에서 부터 구할 수 있다.
- 동적계획법의 구현 방식
  - Top-down : 큰 문제를 작은 문제로 쪼개면서 푼다. 재귀로 구현
  - Bottom-up : 작은 문제부터 차례대로 푼다. 반복문으로 구현
